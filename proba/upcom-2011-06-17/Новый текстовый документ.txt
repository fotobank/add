Этот класс может добавить загруженные файлы в ZIP архив. Он может генерировать HTML для формы, чтобы загрузить файл. класс может обрабатывать запрос при загрузке файла и добавить его в ZIP архив в определенной директории. Он также может генерировать случайное имя файла, поэтому загруженный файл в ZIP архив является уникальным, для того, чтобы избежать перезаписи других файлов в архиве. класса также может читать список архивов ZIP доступны в каталоге выгрузки, удаления ZIP архиве, показать список всех файлов в каждой архив и позволяет удалять отдельные файлы в архиве.
