SZUserMgnt 1.2
About
SZUserMgnt is a smart User Management System with a complete administration area and many features
  • Create user
  • Remove user
  • Edit user
  • Password reminder
  • Activation/inactivation
  • Custom unlimited security Levels
  • SZUserMgnt also comes with authentification functions and different security levels.
SZUserMgnt allows you to write email templates that will be used when you register a new user or send out a password reminder.

Installation instructions
Preperations
  1. Download MySQLHandler and save it to the same folder as the other files
  2. Download SZMail and save it to the same folder as the other files
  3. Import the SQL data
  4. Configure MySQLHandler for your database
  5. Configure SZUserMgnt for your own preferences
  6. You're ready to start! :)
