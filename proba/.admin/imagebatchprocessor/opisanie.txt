Этот пакет может быть использован для применения графических операций на множестве файлов изображений. Предоставляет единый класс, который имеет величины со значениями, которые определяют параметры поддерживаемых графических операций на каждом из файлов, подлежащих обработке. Основной класс может пройти по списку файлов и применять каждую из определенных операций. В настоящее время она может пропорционально изменить размер изображения для создания эскизов, вращать изображения, конвертировать в различные форматы изображений, добавлять текстовые метки или добавлять водяные знаки.
