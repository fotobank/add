PhpFileAdmin - скрипты для удалённого администрирование файлов на сервере

1. Системные требования
  • PHP версии 4 и выше
  • Веб-Сервер Apache с в ключённым .htaccess (рекомендуется)
    Данные возможности доступны на большинстве серверов, поэтому здесь проблем быть не должно (PhpFileAdmin хорошо работает на серверах от Red-Host)

    2. Установка и настройка
    В архиве, который вы скачали есть два файла (install_data.d, install.php)
    Закачайте их в любую папку на сервере и поставьте этой папке права 777 (PHP должно иметь полный доступ к данной папке)
    Затем откройте страницу http://ваш_домен/папка/install.php
    Далее следуйте инструкциям инсталлятора

    3. Ответы на некоторые вопросы

    PhpFileAdnmin не может создавать папки и удалять файлы, что делать?
    Для создания / изменения файлов в папке, пользователю, под чьим именем исполняется PHP скрипт, нужны права на запись/выполнение
    Дайте право записи+выполнения на данную папку через панель управления сервером (обычно ставят 777)
    (Используйте FTP или другое средство управления уровня сервера. Функция смены прав в PhpFileAdmin не сможет изменить права на файл, если у PHP не будет прав)

    Работает ли PhpFileAdmin на Windows системах?
    Да, работает.

    Мне не нравится дизайн, его можно изменить?
    Да, можно. Все шаблоны дизайна лежат в папке tpl (доступна после установки)

    Я написал движок на PHP, можно ли добавить в него PhpFileAdmin?
    Если он предназначен не для коммерческого распространения, то вы смело можете встраивать в него PhpFileAdmin
