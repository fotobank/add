Система статистики на файловой базе Plahov_count_1_0

ВНИМАНИЕ ! 
Данный скрипт расспростроняется бесплатно по лицензии GNU, вы устанавливаете 
его на свой страх и риск, автор не несёт никакой ответственности за ущерб причинённый
вам данным скриптом. Если вы не согласны с этим, то немедленно удалите этот скрипт 
с вашего жёсткого диска. 


Система рассчитана на сбор статистики посещения вашего сайта. 

1. Количество хитов для каждой страницы 
2. Хосты и хиты в отдельно взятый день, а также за периоды 
3. Статистика браузеров и систем посетителей 
4. Статистика ip-адресов посетителей, с возможностью бана 
5. Статистика поисковых роботов 
6. Статистика поисковых запросов 
7. Статистика адресов, с которых перешел посетитель 

Есть возможность вывода на счетчике-картинки хостов и хитов за текущий день, для этого надо, 
чтобы на сервере была установлена библиотека по работе с изображениями GDlib.

Для установки системы нужно распаковать архив в корень сайта, и подключить файл count.php
в те страницы, которые вы хотите, чтобы участвовали в сборе статистики.

Подключение счетчика:

<?
  $title= "Главная страница"; // название страницы
  $path = ""; // путь до корневого каталога
  include($path."count.php"); // подключение файла
?>

Внимание!!!

"Путь до файла" это имеется в виду на сколько уровней надо будет подняться чтобы добраться до корневого каталога
например если такая струкрура сайта:
-------------------------------------------------------------------------
..
папка 1
    | 
   +--- файл 1  // здесь $path = "../";
папка 2
    | 
   +--- файл 2 // здесь $path = "../";
    | 
   +--- файл 3 // здесь $path = "../";
    | 
   +--- папка 3
             | 
            +--- файл 4  // здесь $path = "../../";
папка 4
файл 5  // здесь $path = "";
count.php

-------------------------------------------------------------------------


Для подключение счетчика-картинки достаточно просто в нужном вам месте вставить код

<?
  echo "<img align=left src=".$path."data/img/cntimg.php border=0>";
?>
