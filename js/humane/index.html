Humane JS

Простая, современная, независимая, хорошо протестированная, ненавязчивая система для вывода сообщений.
Использует трансформации CSS, когда они доступны, или анимацию JavaScript. Включает поддержку мобильных систем.

Выберите тему оформления:

Нажмите на коде ниже, чтобы увидеть его в действии:

humane("Добро пожаловать!");
humane.log("Добро пожаловать!");
 
humane.info("Запись <b>392</b> обновлена.");
 
humane.error("Неправильное имя пользователя и/или пароль");
 
humane.success(["Список","разных","пунктов"]);
 
humane("Второй параметр задает возвратную функцию, запускаему по завершении времени вывода", function(){
   document.body.style.backgroundColor="#6699FF";
});

Опции

humane.timeout = (число миллисекунд);

Устанавливает время показа сообщения.

Попробуйте: 2500 (2с - по умолчаниюdefault) 5000 (5с) 500 (0.5с) 

humane.waitForMove = (true|false);

Ожидать события мыши, клавиатуры или сенсорного экрана для скрытия сообщения (после завершения отсчета времени показа)

Попробуйте: true false (по умолчанию) 

humane.forceNew = (true|false);

Выводить новое сообщение немедленно

Попробуйте: true false (по умолчанию) 

humane.clickToClose = (true|false);

Сообщение будет закрываться по нажатию клавиши мыши

Попробуйте: true false (по умолчанию)
