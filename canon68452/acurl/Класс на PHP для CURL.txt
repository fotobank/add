Этот скрипт является оберткой функций CURL расширения для выполнения запросов доступа к HTTP- и FTP-серверам.

Скрипт может выполнять несколько типов операций:

    Пинговать сервер заданного URL, чтобы проверить, отвечает ли он как ожидалось;
    Отправлять запрос на URL и получать заголовки и содержимое;
    Отправлять запрос HTTP аутентификации;
    Посылать HTTP запросы на URL методом POST;
    Загружать файлы через FTP сервер;
    Скачивать файлы с FTP сервера.

Ещё скрипт имеет функцию установки любого значения User Agent для браузера, а также может обрабатывать куки (cookies).
